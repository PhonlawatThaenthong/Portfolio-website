# Phonlawat Thaenthong — Resume Website

Personal resume / portfolio site built with **Next.js 15** + **Tailwind CSS**.

- **Section 1 — Intro:** profile, contact, education, core skills
- **Section 2 — Projects:** timeline of work experience & projects

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
npm start
```

## Deploy to Vercel

```bash
npm i -g vercel
vercel        # preview
vercel --prod # production
```

Or push to GitHub and import the repo at https://vercel.com/new — Vercel
auto-detects Next.js, no config needed.

Edit your details in [`lib/data.js`](lib/data.js).
